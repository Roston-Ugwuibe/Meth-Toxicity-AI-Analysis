# Meth-Toxicity-AI-Analysis
AI-assisted analysis of methamphetamine-induced toxicity using biochemical and histological data
Focus Areas
Antioxidant enzymes (SOD, CAT, GPx)
Heamatological parameters changes (full blood count)
Inflammatory markers (IL-6, IL-1β, TNF-α)
Kidney function parameters (urea, creatinine, Na, K, Cl, HCO3)
Lipid profile changes (Total cholesterol, HDL-C, TAG LDL-C, VLDL-C)  
Liver enzymes markers (Total protein, AST, ALT, ALP, Total Bilirubin)
Neurotransmitter (Dopamine, noradrenaline, serotonin)
Reproductive hormones (Testosterone, Estradiol)
Histological scoring (brain cortex, brain striatrum, heart, Kidney, liver, Ovaries, testes)
Objective
To apply AI and data analysis techniques to identify patterns and potential therapeutic effects of Moringa oleifera extracts.
Tools (Planned)
Python
Excel
Statistical analysis (ANOVA, Tukey)
